## Objetivo
Implementar una aplicacion React que demuestre el uso de:
- Functional Components
- Props y State
- Event Handlers
- Basic Hooks (useState, useEffect)

## Requisitos Cumplidos
- [x] Crear al menos 3 componentes
- [x] Implementar useState para manejo de estado
- [x] Usar props para comunicacion entre componentes
- [x] Manejar eventos onClick y onChange
- [x] Aplicar estilos CSS

Instalacion y Uso
Clonar repositorio

Instalar dependencias con npm install

Ejecutar con npm start

La aplicacion se abrira en http://localhost:3000

Scripts
npm start - Inicia servidor de desarrollo

npm run build - Crea version de produccion

npm test - Ejecuta pruebas unitarias

Tecnologias
React Hooks

CSS Modules

Fetch API

Git para control de versiones
